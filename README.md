# Todo List

Приложение для управления задачами на React 18 + TypeScript.

## Демо

https://todo-app-react-sigma-blond.vercel.app/

## Что умеет

- Добавлять задачи
- Отмечать выполненные
- Удалять задачи
- Сохраняет в localStorage

## Технологии

- React 18
- TypeScript
- Vite

## Запуск

npm install
npm run dev
